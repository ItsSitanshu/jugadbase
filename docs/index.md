# Welcome to Jugadbase

Jugadbase is a simple and fun database designed for quick testing and prototyping. It uses **JQL** (Jugad Query Language) instead of SQL, making it easier to interact with your data in a lightweight way.

## Key Features

- **Easy Setup**: Get up and running in no time without complex configurations.
- **Fast and Lightweight**: Designed for rapid data testing and experiments.
- **JQL (Jugad Query Language)**: Forget SQLâ€”JQL lets you interact with your data in an intuitive and fun way.
- **Minimal Security**: Not meant for production, just testing and playing around with data.
